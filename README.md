# testPB
Repo with test task

В архиве Вы найдете 2 приложения, общую библиотеку для работы с очередью и sql файл для инициализации БД.
Приложение ClientRequestApp отвечает за логирования входящего запроса и отправку в очередь.
Приложение TestTaskPb обрабатывает очередь, сохраняет запросы в бд и содержит API.
